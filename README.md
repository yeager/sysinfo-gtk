# Sysinfo GTK

System information viewer — a modern hardinfo2 alternative.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install sysinfo-gtk
```
